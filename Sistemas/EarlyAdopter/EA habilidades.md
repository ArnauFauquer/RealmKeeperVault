# Habilidades
Las habilidades en el las reglas de [[EarlyAdopter]] funciona de la siguiente manera.

Siempre se usa 1 dado de 20 caras (1d20) para superar las pruebas con sacar un 12 o mas vale.

## Fuerza

Se usa para atacar con armas, empujar, agarrar, etc.

## Agilidad

Se usa para moverte, esquivar, sigilo, etc.

## Presencia

Se usa para apuntar, escuchar, oler, hablar, persuadir, distraer, etc.
